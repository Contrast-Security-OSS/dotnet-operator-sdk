Namespace KubeOps.Operator.Entities.Annotations

Classes

AdditionalPrinterColumnAttribute

Defines a property as an additional printer column.

DescriptionAttribute

Defines a description for a property. This precedes the description found in a XML documentation file.

EmbeddedResourceAttribute

Defines a property as an embedded resource. This property can contain another kubernetes object (e.g. a k8s.Models.V1ConfigMap or a k8s.Models.V1Deployment). This implicitly sets the PreserveUnknownFieldsAttribute.

ExternalDocsAttribute

Defines that the property has an external documentation.

GenericAdditionalPrinterColumnAttribute

Defines a generic additional printer column. With this, other elements (such as Metadata.Name) can be referenced. In contrast to the AdditionalPrinterColumnAttribute, all needed information must be provided.

IgnoreEntityAttribute

Attribute that states that the given entity should be ignored during CRD generation.

ItemsAttribute

Define minimum and maximum items count for an array property.

KubernetesEntityShortNamesAttribute

Define "shortNames" for CRDs.

LengthAttribute

Defines length limits for properties.

MultipleOfAttribute

Defines the factor that a numeric value must adhere to.

PatternAttribute

Define a regex validator for the property.

PreserveUnknownFieldsAttribute

Defines that a property should keep unknown fields so that kubernetes does not purge additional structures.

RangeMaximumAttribute

Defines a range maximum for a numeric property.

RangeMinimumAttribute

Defines a range minimum for a numeric property.

RequiredAttribute

Defines a property of a specification as required.

StorageVersionAttribute

This attribute marks an entity as the storage version of an entity. Only one storage version must be set. If none of the versions define this attribute, the "newest" one is taken according to the kubernetes versioning rules. GA > Beta > Alpha > non versions.
